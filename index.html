Project 2

Firewall for Common Protocols

As part of a networking project, I created a network topology with over 10 devices, including an air-gapped floor, using Python. I implemented a firewall that was capable of blocking TCP, IPV4, and HTTP packets, ensuring the security of the network. To confirm the effectiveness of the firewall, I utilized Wireshark to monitor network traffic and ensure that no packets from TCP, IPV4, or HTTP sources were received. I also performed tests using Mininet to simulate network conditions and verify that the firewall was functioning correctly. Overall, the project demonstrated my ability to design and implement a secure network using Python, as well as my proficiency with tools like Wireshark and Mininet.
